# Самостоятельный проект интернет магазина по продаже яблочной продукции

## Описание:
"Yablochki" - это интернет магазин яблочной продукции.

Самостоятельно разработанный макет: [Figma](https://www.figma.com/file/GcxJVIC25R1hEBxz0aEfFA/Yablochki?node-id=1%3A2)

## Технологии:
* Next.JS
* Tailwind
* Strapi
* DaisyUI

First, run the development server:

Для запуска проекта:

```bash
1) yarn install
2) yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Откройте [http://localhost:3000](http://localhost:3000) в браузере для того чтобы увидеть результат.
